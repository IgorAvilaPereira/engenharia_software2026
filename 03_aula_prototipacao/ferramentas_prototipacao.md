# Ferramentas Gratuitas e Open Source para Prototipação de Telas

## Objetivo

Este documento apresenta ferramentas gratuitas e de código aberto (open source) que podem ser utilizadas para prototipação de interfaces (wireframes e mockups), especialmente em contextos acadêmicos e de desenvolvimento de software.

---

## 1. Penpot

### Descrição

Ferramenta moderna de design e prototipação colaborativa baseada na web.

### Principais Características

* 100% open source
* Funciona no navegador
* Interface semelhante ao Figma
* Suporte a prototipação interativa
* Componentes reutilizáveis
* Colaboração em tempo real
* Possibilidade de auto-hospedagem

### Vantagens

* Ferramenta completa e atual
* Sem limitações significativas no uso gratuito
* Ideal para ensino e projetos reais

### Desvantagens

* Pode exigir curva de aprendizado inicial

---

## 2. Pencil Project

### Descrição

Ferramenta desktop para criação de wireframes e protótipos simples.

### Principais Características

* Open source
* Funciona offline
* Interface simples
* Biblioteca de componentes básicos

### Vantagens

* Muito fácil de usar
* Leve e rápida
* Ideal para iniciantes

### Desvantagens

* Interface desatualizada
* Não possui colaboração em tempo real
* Recursos limitados para protótipos avançados

---

## 3. Akira

### Descrição

Ferramenta de design voltada principalmente para sistemas Linux.

### Principais Características

* Open source
* Foco em design de interfaces
* Aplicação desktop

### Vantagens

* Integração com ambiente Linux
* Projeto promissor

### Desvantagens

* Ainda em desenvolvimento
* Poucos recursos comparado a alternativas
* Comunidade menor

---

## 4. Quant-UX

### Descrição

Ferramenta open source voltada para prototipação e testes de usabilidade.

### Principais Características

* Open source
* Prototipação interativa
* Testes de usabilidade integrados
* Execução via navegador

### Vantagens

* Permite validar protótipos com usuários
* Abordagem focada em UX

### Desvantagens

* Interface menos intuitiva
* Menos popular

---

## 5. Wireflow

### Descrição

Ferramenta open source focada em fluxos de usuário e wireframes.

### Principais Características

* Open source
* Interface baseada em blocos
* Foco em fluxos de navegação

### Vantagens

* Excelente para mapear jornadas de usuário
* Simples e direto

### Desvantagens

* Pouco foco em design visual
* Recursos limitados para prototipação detalhada

---

## Comparação Geral

| Ferramenta     | Tipo    | Colaboração | Nível    | Indicação Principal |
| -------------- | ------- | ----------- | -------- | ------------------- |
| Penpot         | Web     | Sim         | Avançado | Projetos completos  |
| Pencil Project | Desktop | Não         | Básico   | Ensino inicial      |
| Akira          | Desktop | Não         | Básico   | Usuários Linux      |
| Quant-UX       | Web     | Parcial     | Médio    | Testes de UX        |
| Wireflow       | Web     | Não         | Básico   | Fluxos de usuário   |

## 6. Ferramentas Gratuitas (com plano free) e Proprietárias

### Figma

#### Descrição

Ferramenta amplamente utilizada para design de interfaces e prototipação colaborativa.

#### Principais Características

* Baseada na web
* Colaboração em tempo real
* Protótipos interativos
* Componentes reutilizáveis

#### Vantagens

* Muito completa
* Padrão de mercado
* Plano gratuito bastante utilizável

#### Desvantagens

* Não é open source
* Algumas limitações no plano free

---

### Balsamiq

#### Descrição

Ferramenta focada em wireframes de baixa fidelidade.

#### Principais Características

* Interface simples
* Estilo desenhado à mão

#### Vantagens

* Muito fácil de usar
* Ideal para iniciantes

#### Desvantagens

* Limitado visualmente
* Não é open source

---

### Adobe XD

#### Descrição

Ferramenta de design de interfaces da Adobe.

#### Principais Características

* Protótipos interativos
* Integração com outras ferramentas Adobe

#### Vantagens

* Interface amigável
* Recursos profissionais

#### Desvantagens

* Não é open source
* Plano gratuito limitado

---

### Canva

#### Descrição

Plataforma de design simplificada com suporte a prototipação básica.

#### Principais Características

* Templates prontos
* Interface extremamente simples

#### Vantagens

* Muito fácil de usar
* Ideal para protótipos rápidos

#### Desvantagens

* Pouco técnico
* Limitado para sistemas complexos

---

### MockFlow

#### Descrição

Ferramenta online para criação rápida de wireframes.

#### Principais Características

* Interface simples
* Foco em prototipação rápida

#### Vantagens

* Fácil de usar
* Rápido para ideias iniciais

#### Desvantagens

* Muitas limitações no plano gratuito
* Não é open source

---

## Conclusão

Para a maioria dos cenários, especialmente em ensino e desenvolvimento de sistemas completos, o Penpot é a melhor escolha entre as ferramentas open source.

Ferramentas como Pencil Project e Wireflow são úteis em etapas iniciais de concepção, enquanto Quant-UX pode ser utilizado para validação com usuários.

A escolha deve considerar o nível de complexidade do projeto e a necessidade de colaboração.

---

## Recomendação Final

* Uso geral (ensino + prática profissional): Penpot
* Wireframe simples: Pencil Project
* Fluxos de usuário: Wireflow
* Testes de usabilidade: Quant-UX
