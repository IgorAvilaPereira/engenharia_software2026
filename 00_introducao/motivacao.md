# Introdução

software não é só código.
programar sem organização causa problemas.
    >> A engenharia de software surge para organizar o desenvolvimento.
    

## O ciclo de vida do software

Etapas comuns:

* Levantamento de requisitos
entender o que o usuário precisa

* Projeto (design)
planejar como o sistema será

* Implementação
escrever o código

* Testes
verificar se funciona

* Manutenção
corrigir e melhorar o sistema
