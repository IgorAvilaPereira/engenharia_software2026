1. Como será o processo de criação de conta?

   * Comprar jogos?
   * Baixar jogos?
   * Atualizar jogos automaticamente?
3. Haverá biblioteca pessoal de jogos?

   * Avaliar jogos?
   * Escrever reviews?
   * Dar notas?
5. Existirá lista de desejos (wishlist)?
6. Os usuários poderão solicitar reembolso?
7. Haverá sistema de conquistas (achievements)?

# 5. Funcionalidades sociais

1. Usuários poderão adicionar amigos?

# 6. Funcionalidades para desenvolvedores
